## Units
- g	= gram
- mg	= milligram
- mcg	= microgram
- U	= unit
- TU	= thousand units
- MU	= million units
- mmol	= millimole
- ml	= milliliter (e.g. eyedrops)

## Route of administration (Adm.R)
- Implant	= Implant
- Inhal	= Inhalation
- Instill	= Instillation
- N	= nasal
- O	= oral
- P	= parenteral
- R	= rectal
- SL	= sublingual/buccal/oromucosal
- TD	= transdermal
- V	= vaginal
